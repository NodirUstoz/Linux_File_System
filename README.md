## üìã Table of Contents

Each guide covers the following topics:

1. **Introduction and Basic Concepts**
   - What is Linux?
   - Understanding CTF competitions
   - Terminal and Shell fundamentals

2. **Linux File System Architecture**
   - "Everything is a file" philosophy
   - Filesystem Hierarchy Standard (FHS)
   - File types and inodes

3. **Navigation and Basic Commands**
   - Directory navigation (`cd`, `pwd`)
   - File listing (`ls`)
   - User information (`whoami`, `id`)

4. **File Search Methods**
   - Powerful `find` command
   - Fast `locate` utility
   - Pattern matching with globbing

5. **Reading File Contents**
   - `cat`, `less`, `more`
   - `head` and `tail`
   - Binary file analysis with `strings`

6. **Getting Help**
   - `--help` flag
   - `man` pages
   - `info` documentation

7. **Text Editors**
   - `nano` for beginners
   - `vim` for advanced users

8. **Permissions System**
   - Understanding file permissions
   - Binary number system and `chmod`
   - SUID, SGID, and sticky bits

9. **Important Directories**
   - `/etc` - Configuration files
   - `/home` - User directories
   - `/var` - Variable data
   - `/proc` - Process information

10. **Working with Archives**
    - `tar` archives
    - `zip`/`unzip`
    - Compression utilities

11. **Disk Operations**
    - `df` - Disk free space
    - `du` - Disk usage
    - Mount operations

12. **Symbolic Links**
    - Hard links vs symbolic links
    - Creating and managing symlinks

13. **Practical Exercises**
    - Hands-on challenges
    - Step-by-step solutions

14. **CTF Strategies**
    - First steps in CTF challenges
    - Flag hunting techniques
    - Common pitfalls to avoid
